# Lap 4 Code Challenge

## Installation and Usage
To use deployed app:

Visit: https://ltl-url.herokuapp.com/

To use local version:
- clone and cd into repo folder
- `pipenv shell`
- `pipenv install`
- `pipenv run dev`
- Available on http://127.0.0.1:5000
